Vision Transformer: An Image is Worth 16x16 Words

How pure Transformers revolutionized computer vision and why scale matters

Vo Hoang Nhat Khang (Chris) PhD Student in NLP at MBZUAI
January 15, 2025 12 min read
Vision Transformer Computer Vision Transformers Image Classification

Overview and Key Insight

This post is my analysis of An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale (Dosovitskiy et al., 2021). The core breakthrough is elegantly simple: treat image patches as "words" and apply a standard Transformer directly to vision. This paper fundamentally challenged the dominance of CNNs in computer vision by showing that pure attention mechanisms can achieve state-of-the-art results when trained at sufficient scale.

The Motivation: Why Move Beyond CNNs?

Before ViT, computer vision was dominated by convolutional architectures. While Transformers had revolutionized NLP, their application to vision remained limited. Previous attempts either used attention alongside CNNs or replaced specific CNN components while keeping the overall structure. ViT's radical approach was to eliminate CNNs entirely and apply a pure Transformer directly to image patches.

The key insight: Transformers lack the inductive biases inherent in CNNs (translation equivariance, locality), but large-scale training can overcome these limitations. This is a classic example of "scale beats inductive bias" in deep learning.

Architecture: From Images to Sequences

ViT's architecture is remarkably straightforward. Given an image \(x \in \mathbb{R}^{H \times W \times C}\), the process is:

  1. Patch Embedding: Split image into patches of size \(P \times P\) (typically 16×16), resulting in \(N = HW/P^2\) patches
  2. Linear Projection: Flatten each patch and project to dimension \(D\): \(x_p \in \mathbb{R}^{P^2C} \rightarrow z_0 \in \mathbb{R}^D\)
  3. Position Embedding: Add learnable position embeddings to retain spatial information
  4. Classification Token: Prepend a learnable [CLS] token for classification
  5. Transformer Encoder: Apply standard Transformer blocks

The input sequence is: \(\mathbf{z}_0 = [\mathbf{x}_{\text{class}}; \mathbf{x}_p^1\mathbf{E}; \mathbf{x}_p^2\mathbf{E}; \ldots; \mathbf{x}_p^N\mathbf{E}] + \mathbf{E}_{\text{pos}}\), where \(\mathbf{E}\) is the patch embedding projection and \(\mathbf{E}_{\text{pos}}\) are position embeddings.

Transformer Encoder Details

Each Transformer block consists of:

\[ \mathbf{z}'_l = \text{MSA}(\text{LN}(\mathbf{z}_{l-1})) + \mathbf{z}_{l-1} \] \[ \mathbf{z}_l = \text{MLP}(\text{LN}(\mathbf{z}'_l)) + \mathbf{z}'_l \]

Where MSA is Multi-Head Self-Attention, LN is Layer Normalization, and MLP is a two-layer feed-forward network. The final classification head is a simple linear layer applied to the [CLS] token.

Training Strategy and Data Requirements

The paper demonstrates a crucial finding: ViT's performance heavily depends on training data scale. When trained on mid-sized datasets like ImageNet without strong regularization, ViT performs modestly compared to ResNets. However, when pre-trained on large datasets (14M-300M images), ViT achieves excellent results.

Key training details:

  • Pre-training: ImageNet-21k (14M images) or JFT-300M (300M images)
  • Fine-tuning: Smaller datasets with higher resolution
  • Regularization: Dropout, weight decay, and data augmentation
  • Optimization: Adam with cosine learning rate schedule

Key Results and Performance

ViT achieved remarkable results across multiple benchmarks:

  • ImageNet: 88.55% top-1 accuracy (ViT-H/14)
  • ImageNet-ReaL: 90.72% top-1 accuracy
  • CIFAR-100: 94.55% accuracy
  • VTAB-19: 77.63% average accuracy

Understanding ViT's Attention

The paper provides fascinating insights into how ViT processes images through attention:

  • Attention Distance: Lower layers attend locally, higher layers attend globally across the image
  • Head Diversity: Different attention heads learn to focus on different spatial regions and features
  • Class Token: The [CLS] token aggregates information from all patches for final classification

ViT vs CNNs: The Trade-offs

ViT Advantages:

  • Global receptive field from the first layer
  • Better scalability with data and model size
  • More interpretable attention patterns
  • Unified architecture for vision and language

CNN Advantages:

  • Strong inductive biases (translation equivariance, locality)
  • Better performance on small datasets
  • More efficient for high-resolution images
  • Proven track record in production systems

Limitations and Challenges

  • Data Dependency: Requires large-scale pre-training to overcome lack of inductive biases
  • Computational Cost: Quadratic complexity with respect to image size
  • Position Encoding: Fixed position encodings may not generalize well to different resolutions
  • Local Features: May struggle with fine-grained local patterns that CNNs excel at

Mathematical Formulation

The multi-head self-attention mechanism in ViT is:

\[ \text{Attention}(\mathbf{Q}, \mathbf{K}, \mathbf{V}) = \text{softmax}\left(\frac{\mathbf{Q}\mathbf{K}^T}{\sqrt{d_k}}\right)\mathbf{V} \]

Where \(\mathbf{Q}\), \(\mathbf{K}\), \(\mathbf{V}\) are query, key, and value matrices, and \(d_k\) is the dimension of the key vectors. The multi-head attention concatenates multiple attention heads:

\[ \text{MSA}(\mathbf{x}) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h)\mathbf{W}^O \]

Practical Implementation Tips

  • Patch Size: Smaller patches (16×16) generally work better than larger ones
  • Position Encoding: Learnable position embeddings often outperform fixed sinusoidal ones
  • Resolution: Fine-tune at higher resolution than pre-training for better performance
  • Regularization: Use appropriate dropout and weight decay to prevent overfitting

Impact and Legacy

ViT fundamentally changed computer vision by demonstrating that pure attention mechanisms could compete with and surpass CNNs. This work paved the way for:

  • Vision-language models (CLIP, DALL-E, GPT-4V)
  • Unified architectures across modalities
  • Large-scale vision models (PaLI, PaLM-E)
  • Efficient variants (Swin Transformer, PVT, etc.)

Future Directions

The success of ViT opened several research directions:

  • Efficiency: Reducing computational complexity for real-time applications
  • Multimodal: Extending to video, 3D, and other modalities
  • Architecture: Exploring hybrid CNN-Transformer models
  • Training: Self-supervised and few-shot learning approaches

Citation

Dosovitskiy, A., Beyer, L., Kolesnikov, A., et al. "An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale." ICLR 2021. Available at https://arxiv.org/abs/2010.11929.
